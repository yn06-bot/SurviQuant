# Analysis & Dashboard Skills Index

이 문서는 S&P 500 생존분석 AI 대시보드를 구동하기 위한 핵심 규칙들의 진입점입니다. 
시스템의 유연한 확장과 유지보수를 위해 범용적인 데이터/시각화 규칙(Core)과 특정 금융 분석 논리가 담긴 도메인 규칙(Domain), 그리고 자동화 운영 규칙(Ops)을 분리하여 관리합니다.

### 1. Core Rules (01_core_rules.md)

1. 어떤 금융 데이터가 들어와도 일관되게 처리할 수 있는 범용 데이터 스키마(OHLCV)와 기술적 지표 계산식을 정의합니다.

2. 대시보드 화면에서 차트와 보조지표가 렌더링되는 기준(오버레이 vs 서브차트)을 규정합니다.

### 2. Domain Scoring Rules (02_domain_scoring.md)

1. 단순 가격 예측이 아닌 생존분석(Survival Analysis)을 적용하기 위한 사건(Event) 정의와 모델링 로직을 담고 있습니다.

2. 모델의 예측 결과를 100점 만점의 'AI Score'로 변환하고, 투자 성향에 따라 점수 가중치를 다르게 부여하는 핵심 알고리즘을 명세합니다.

### 3. Operations Pipeline (03_ops_pipeline.md)

1. 대시보드의 실시간성을 보장하기 위해 매일 최신 데이터를 결합하고 갱신하는 방식을 정의합니다.

2. 자동화된 워크플로우(GitHub Actions) 설계와 생존분석 모델 재학습 주기에 대한 운영 기준을 제공합니다.
